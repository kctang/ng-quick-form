ng-quick-form

Quick form was designed to let Angular developers declare form fields to collect data for common use cases easily.

Where possible, it uses Angular's form capabilities (e.g. validation, reactive form capabilities) to retain familiarity.

Getting Started
  • This library was designed for Angular 7+ and uses Angular Material to display form fields, so you will need a project with these dependencies configured.
  • To install this library from NPM:
    npm install ng-quick-form
  • Understand the key concepts listed below, review the demo and code.
  • Check out the demos, and start tweaking them in your projects.
Key Concepts:
  1. QuickFormModule
    Register this module in your Angular application.
  2. QuickFormField
    Definition for a form field. A form basically constructed from an array of fields. Granted, this approach has limitations when it comes to complex forms. However, that is not the scope of this library. The objective is to make typical form usage really really simple. For the rest, you can always fallback to Angular's FormBuilder.
  3. QuickForm.makeForm(fields: QuickFormField[]): FormGroup
    This function is used to create a standard Angular FormGroup based on an array of field definitions.
  4. <quick-form-field>
    This component is used to render form fields. Typical usage is to iterate through form fields and render each field. 
    <quick-form-field *ngFor="let field of fields" [field]="field" [form]="form"> </quick-form-field>
  5. QuickForm.preProcessFormValues(formValues: any)
    Use this function to convert form values to a structure based on field definitions. It should be called before processing form's JSON data.
    Why is this needed? Some field types are rendered as multiple form controls. For example, checkbox type is represented as a FormArray with multiple FormGroups. The main FormGroup's value contains values from these intermediate controls.
