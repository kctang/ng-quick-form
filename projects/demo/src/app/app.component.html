ng-quick-form
Simplify usage of Angular forms with Angular Material for common use cases.
Instead of using FormGroup, FormArray and FormControl objects directly to define form structure, this library uses QuickFormField to declare form fields. It provides <quick-form-field> component to render fields, complete with validation and error message handling.
Supported field types:
  • autocomplete
    A text field with autocomplete with a list of options.
    • Supports dynamic options based on current form value via optionsFn.
  • checkbox
    A field that displays multiple checkboxes, allowing user to select one or more option. Result of selection is returned as a single array value (instead of one value per checkbox control).
  • chips
    A field that display a list of values as individual, keyboard accessible, chips. While users can type any chip value, it also supports auto-completion via a list of options. In addition, the options can be categorized/grouped (see demo).
  • password
    A password field.
  • radio
    A field that renders radio buttons based on options specified.
  • select
    A field that lets user select a value based on a list of options.
    • Supports multiple selection via the selectMultiple property.
    • Supports dynamic options based on current form value via optionsFn.
  • separator
    This is not a field to collect data. It displays a separator using the field title.
  • switch
    A field that displays an Angular Material "slide toggle" control.
  • text
    A text field.
  • textarea
    A text field that supports multiple lines.
Getting Started
  • This library was designed for Angular 8+ and uses Angular Material to display form fields, so you will need a project with these dependencies configured.
  • To install this library from NPM:
    npm install ng-quick-form
  • Understand the key concepts listed below, review the demo and code.
  • Check out the demos, and start tweaking them in your projects.
Key Concepts:
  1. QuickFormModule
    Register this module in your Angular application.
  2. QuickFormField, QuickFormFieldType, QuickFormFieldOptionDefinition
    Definition for a form field. A form basically constructed from an array of fields. Granted, this approach has limitations when it comes to complex forms. However, that is not the scope of this library. The objective is to make typical form usage really really simple. For the rest, you can always fallback to Angular's FormBuilder.
  3. QuickForm.makeForm(fields: QuickFormField[]): FormGroup
    This function is used to create a standard Angular FormGroup based on an array of field definitions.
  4. <quick-form-field>
    This component is used to render form fields. Typical usage is to iterate through form fields and render each field. 
    <quick-form-field *ngFor="let field of fields" [field]="field" [form]="form"> </quick-form-field>
  5. QuickForm.preProcessFormValues(formValues: any)
    Use this function to convert form values to a structure based on field definitions. It should be called before processing form's JSON data.
    Why is this needed? Some field types are rendered as multiple form controls. For example, checkbox type is represented as a FormArray with multiple FormGroups. The main FormGroup's value contains values from these intermediate controls.
